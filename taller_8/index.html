Esquema de como acceder a un Nodo

Pasos para crear un nodo tipo element

  • Se crea un nodo de tipo Element
  • Utlizamos el método createElement
  • las sintasis es la siguiente: var nombre_elemento =document.createElement("p")
  • acabamos de crear un elemento p que va dentro del metodo.


    • Pasos para eliminar un nodo tipo element

  • Se utiliza en método removeChild()
  • Dentro del paréntesis de método ponemos el nodo que queremos eliminar
  • Debemos eliminarlo desde su nodo padre Ejemplo:
  • suprimir = document.getElementById("provisional")
    suprimir.parentNode.removeChild(suprimir)
